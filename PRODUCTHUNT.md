# ProductHunt Launch — Solar System Agents

## Listing Details

**Name:** Solar System Agents

**Tagline (60 chars max):**
Cyberpunk mission control for your AI agent fleet

**Description:**
Solar System Agents is an open-source dashboard suite for monitoring AI agent fleets. Two stunning views:

**Mission Control** — A cyberpunk command center with agent roster, cron job monitoring, system vitals (RAM gauges, donut charts), KPI streams, model provider status, scrolling ticker, and a particle network background with CRT scanline overlay. Real-time status updates via API polling.

**Solar System** — An interactive orbital visualization where each agent is a planet. Size reflects importance, orbital speed reflects activity. Moons for sub-agents, Saturn-style rings for cron jobs. Canvas2D at 60fps.

The whole thing is static HTML with zero dependencies. Clone the repo, open a file, done. Edit config.js to set up your own agents in 30 seconds. Demo mode works out of the box with 12 sample agents.

Works with any multi-agent framework: CrewAI, LangChain, AutoGPT, OpenAI Assistants, Semantic Kernel, or custom setups. Deploy to Vercel or Netlify with one click.

**Key Features:**
- Cyberpunk Mission Control dashboard with glassmorphism panels
- Interactive solar system with orbital mechanics
- Agent roster with live status dots, model info, skill counts
- Cron job monitoring with OK/ERR visual cards
- System vitals with Chart.js gauges (RAM, uptime)
- KPI streams & revenue tracking panel
- Channel status (Slack, Telegram, WhatsApp, API)
- Model provider status (OpenAI, Anthropic, Groq, etc.)
- Animated particle network background (tsParticles)
- CRT scanline overlay & animated ticker bar
- Config-driven: edit one JS file to customize everything
- Demo mode: works instantly with realistic sample data
- Live API polling for real-time agent status
- Mobile responsive
- Zero dependencies — static HTML files
- MIT licensed
